./usb2gpio.py 参数 指令

set - 设置引脚状态
get - 获取引脚状态

./usb2gpio.py set 1 1
设置引脚状态1为高电平
./usb2gpio.py set 1 0
设置引脚状态1为低电平
./usb2gpio.py set 1 1 2 1
设置引脚状态1和2为高电平
./usb2gpio.py set 1 1 2 1 3 1 4 1 5 0 6 0 7 0
设置1/2/3/4为高电平，5/6/7为低电平

fast_set - 串行设置引脚状态

./usb2gpio.py fast_set 1 1 2 1 3 1 4 1 5 0 6 0 7 0
串行设置引脚状态1/2/3/4为高电平，5/6/7为低电平，但是代码是以最快的串行速度一个一个发送出去的
这个功能可以以最快的速度来控制显示设备
